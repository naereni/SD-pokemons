# SD-pokemons
Finetune SD-1.5 to gen new pokemons

## Install


<details>
  <summary>XFormers on macos</summary>
  
```bash
brew install libomp
export PATH="/opt/homebrew/opt/libomp/bin:$PATH"
export LDFLAGS/CPPFLAGS # get while installing
brew install llvm
export LDFLAGS/CPPFLAGS
```
